

# DoubanTweak

![Platform](https://img.shields.io/badge/platform-iOS%2CiPadOS-blue.svg)
[![Release](https://img.shields.io/github/v/release/TouchFriend/DoubanTweak?color=brightgreen)](https://github.com/TouchFriend/DoubanTweak/releases)
![Support](https://img.shields.io/badge/support-douban%207.97.0+-blue.svg)

一个基于 **iOS 逆向分析** 的 **豆瓣无广告越狱插件**，通过修改客户端本地行为移除广告。

> ⚠️ 本项目仅适用于 **已越狱设备**

---

## 📌 项目说明

- 适用于 **已越狱的 iOS / iPadOS 设备**
- 以 **Theos / MonkeyDev** 为基础进行开发
- 仅修改客户端本地逻辑，不涉及任何服务器或接口行为

如果你使用的是 **非越狱设备**，请参考以下项目：

👉 [无广告版豆瓣（非越狱版）](https://github.com/TouchFriend/DoubanMApp)

---

## ✨ 功能特性

- 🚫 去除豆瓣客户端中的绝大多数广告
- 🧹 精简界面，提升浏览体验
- 📱 不影响原有功能与使用流程

---

## 🚀 快速安装（推荐）

在 **越狱设备** 上打开 `Cydia` 应用：

1. 进入 `软件源` → `编辑` → `添加`
2. 添加以下软件源地址：

```text
https://touchfriend.github.io
```

1. 确认列表中出现 **TouchFriend's Repo**
2. 打开 `TouchFriend's Repo` → 插件
3. 安装 **DoubanTweak**
4. 重启 App（或 Respring）

------

## 🛠️ 从源码构建（开发者）

### 1️⃣ 环境准备

- 安装 [MonkeyDev](https://github.com/AloneMonkey/MonkeyDev)
- 确保设备已越狱并可正常调试

------

### 2️⃣ 获取源码

```bash
git clone https://github.com/TouchFriend/DoubanTweak.git
```

------

### 3️⃣ 编译并运行

1. 使用 Xcode 打开工程
2. 选择越狱真机设备
3. 编译并运行
4. 重启豆瓣 App 生效
